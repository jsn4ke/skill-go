## Context

spells.csv 有5个技能，其中冰霜新星(27088)和奥术智慧(27126)未完整实现（缺少 VFX 验证），冲锋(100)已完整实现含位移+眩晕+产怒+伤害。

## Goals / Non-Goals

**Goals:**
- 清理未实现的冰霜新星和奥术智慧配置数据
- 确保清理后所有测试通过
- 确保已实现的火球术和冲锋不受影响

**Non-Goals:**
- 不修改任何 Go 代码逻辑
- 不修改前端代码
- 不清理归档中的历史记录

## Decisions

1. 直接删除 CSV 行，不需要 soft-delete 或注释
2. 更新 loader_test.go 中引用了27088的测试用例，改用已实现的38692数据
3. spell_effects.csv 中冰霜新星有两行效果(0和1)，全部删除
4. 奥术智慧只有一行效果(0)，删除该行

## Risks / Trade-offs

- 风险：删除后重新添加需要重新查 TBC 数据（但 skill.md 有参考数据）
- trade-off：冰霜新星作为 AoE 技能后续可能有用，但现在清理保持代码整洁
